Import Other Translations

If you want to export Other Translations data, you can do it quickly by clicking on or .
Drag and drop file here or click to upload
Choose a file with following extensions: csv, xls, xlsx.
The number of rows to be imported at a time is defined by the chunk size. Increase this value if you have a large file and data is imported very fast. Decrease this value if you encounter memory limits or gateway timeout issues when importing data.
