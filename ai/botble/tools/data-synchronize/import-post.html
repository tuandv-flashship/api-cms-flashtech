Import Posts

If you want to export Posts data, you can do it quickly by clicking on or .
Drag and drop file here or click to upload
Choose a file with following extensions: csv, xls, xlsx.
The number of rows to be imported at a time is defined by the chunk size. Increase this value if you have a large file and data is imported very fast.     Rules

    
    
    
            
    
    
    
    Name

                        
    The Name field accepts a string value of up to 250 characters or may be left blank.


    
                                    
    
    
    Slug

                        
    The Slug field accepts a string value of up to 250 characters or may be left blank.


    
                                    
    
    
    Description

                        
    The Description field accepts a string value of up to 400 characters or may be left blank.


    
                                    
    
    
    Content

                        
    The Content field accepts a string value of up to 300,000 characters or may be left blank.


    
                                    
    
    
    Tags

                        
    The Tags field accepts an array value or may be left blank.


    
                                    
    
    
    Categories

                        
    The Categories field accepts an array value or may be left blank.


    
                                    
    
    
    Status

                        
    Status must be one of the following values: published, draft, pending.


    
                                    
    
    
    Is Featured

                        
    Is featured must be one of the following values: Yes, No.


    
                                    
    
    
    Image

                        
    The Image field accepts a string value or may be left blank.


    
                                    
    
    
    Format Type

                        
    The Format type field accepts a string value of up to 50 characters or may be left blank.
